# Cybersecurity_Awareness_Chatbot


# Project Overview

SecureBot is a console-based chatbot developed in C#.
Its purpose is to educate users about cybersecurity topics such as password safety, phishing, and safe browsing.

The chatbot provides an interactive experience using voice, visuals, and a menu-driven system.



# Features

# Voice Greeting

* Plays a welcome audio message when the program starts
* Uses a `.wav` file with `SoundPlayer`

# ASCII Logo Display

* Displays a cybersecurity-themed logo in ASCII format
* Enhances visual appearance in the console

# User Interaction

* Prompts the user to enter their name
* Uses a **do-while loop** to validate input
* Prevents empty or invalid names
* Personalises chatbot responses using the user’s name

# Chatbot System

* Responds to cybersecurity-related questions
* Uses keyword detection for topics like:

  * Password security
  * Phishing attacks
  * Safe browsing
  * Cyber threats
* Provides a default response for unknown inputs

# Menu System

* Allows users to navigate through options:

  1. Chat with SecureBot
  2. About SecureBot
  3. Help
  4. Exit
* Implemented using a **do-while loop**

# Input Validation

* Handles invalid menu inputs
* Prevents empty chatbot input
* Provides user-friendly error messages

# Console UI Design

* Uses colours, spacing, and borders
* Improves readability and user experience



# Technologies Used

* C#
* .NET Console Application
* System.Media (for audio)
* System.Drawing (for ASCII image)

# How to Run the Program

1. Open the project in Visual Studio
2. Ensure the following files are in the correct directory:

   * `C.A.C_greet.wav`
   * `logo.jpeg`
3. Build and run the program
4. Follow the on-screen instructions



# How to Use

* Enter your name when prompted
* Choose an option from the menu
* Ask questions about cybersecurity
* Type `exit` to return to the menu
